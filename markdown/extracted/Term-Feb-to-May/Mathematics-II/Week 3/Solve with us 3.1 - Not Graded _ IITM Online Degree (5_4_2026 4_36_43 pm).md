$\\textbf{ Definition of a vector space V: }$  

*   A vector space $V$ over $\\mathbb{R}$ is a set along with two functions:

  

1.  $+: V \\times V \\rightarrow V$ ( $V$ is closed under vector addition i.e., if $x, y \\in V \\implies x+ y \\in V, ~\\forall~ x, y \\in V$)
2.  $.: \\mathbb{R} \\times V \\rightarrow V$ ( $V$ is closed under scalar multiplication i.e., if $x \\in V \\text{ and } c \\in \\mathbb{R} \\implies c.x \\in V, ~\\forall~ x \\in V, \\text{ and } \\forall c \\in \\mathbb{R}$)

*    Axioms of vector additions of a vector space

1.  $v\_1 + v\_2 = v\_2 + v\_1, ~\\forall ~v\_1, v\_2 \\in V$.
2.  $v\_1 + (v\_2 + v\_3) = (v\_1 + v\_2) +v\_3, ~\\forall~ v\_1, v\_2 \\text{ and } v\_3\\in V$.
3.  There exists an element 0 (called the zero vector of $V$) in $V$ such that $0+v = v, ~ \\forall~ v\\in V$.
4.  There exists a vector $v'$ in $V$ such that $v'+v = v+ v' = 0, ~ \\forall~ v\\in V$ ($v'$ is called the inverse ( or negative ) of $v$ with respect to addition).

  

*   Axioms relating vector addition and scalar multiplication of a vector space (since this is usual multiplication in real numbers, we suppress the "." sign ).

1.  For each vector $v \\in V$, $1v = v.$
2.  For each vector $v \\in V$ and for each pair $a, b \\in \\mathbb{R}, (a+b)v = av+bv.$
3.  For each $a \\in \\mathbb{R}$ and for each pair $v\_1, v\_2 \\in V, a(v\_1+v\_2) = av\_1+av\_2.$
4.  For each vector $v \\in V$ and for each pair $a, b \\in \\mathbb{R}, (ab)v = a(bv).$

*   Criteria to check a subset $W$ of a vector space $V$ is a subspace:

1.  Zero vector should be in $W$ i.e., $0 \\in W$.

3.  Let $c \\in \\mathbb{R}$ and if $v \\in W$, then $cv \\in W$
